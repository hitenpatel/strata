# Strata

An accessibility-first web component library with a neutral, themeable
baseline — two-tier design tokens and Lit 3 web components. One core,
consumable from plain HTML, React and Vue.

- **Source**: [github.com/hitenpatel/strata](https://github.com/hitenpatel/strata)
- **Package**: `@hitenpatel/strata`

## Philosophy

Strata handles the hard parts — keyboard interaction, ARIA semantics, focus
management, contrast-safe defaults — and stays out of the way on the visual
identity:

- **Accessibility first.** Every component ships WCAG 2.2 AA out of the box:
  visible focus rings, 4.5:1 text contrast in both themes,
  `prefers-reduced-motion` support, and axe-verified stories.
- **Neutral baseline.** A quiet, hairline-and-soft-shadow default look (in the
  spirit of shadcn/ui) that works as-is and doesn't fight your brand.
- **Styling belongs to the implementor.** Two composable theming layers —
  design tokens for the broad strokes, CSS `::part()` for surgical restyling.
  See the [Theming](?path=/docs/docs-theming--docs) page.

## Installation

```bash
npm install @hitenpatel/strata
```

```js
import '@hitenpatel/strata';            // registers all <strata-*> elements
import '@hitenpatel/strata/tokens.css'; // design token custom properties
```

```html
<strata-button variant="primary">Save changes</strata-button>
```

## Token architecture

Tokens live in a single [W3C Design Tokens](https://design-tokens.github.io/community-group/format/)
file and compile to CSS custom properties in two tiers:

1. **Primitives** — raw scales: colour ramps (`--strata-color-blue-600`),
   spacing (`--strata-space-4`), radii, shadows, motion.
2. **Semantic** — role-based aliases components actually consume:
   `--strata-surface`, `--strata-text-muted`, `--strata-accent`,
   `--strata-focus-ring`…

Components only ever reference semantic tokens, which is what makes theming
free.

## Theming

Two attributes on `<html>` switch the bundled token sets; no component code
changes:

```html
<html data-theme="dark">                  <!-- dark theme -->
<html data-variant="showcase">            <!-- violet accent -->
<html data-theme="dark" data-variant="showcase">
```

Try it live: use the **Theme** and **Accent** dropdowns in the toolbar above —
every story responds instantly. For custom themes and per-component
restyling, see [Theming](?path=/docs/docs-theming--docs).

## Framework support

The Lit components are the single source of truth. Thin, typed adapters are
provided for React (`@hitenpatel/strata/react`) and Vue
(`@hitenpatel/strata/vue`) — see the dedicated docs pages in this section.
